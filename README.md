## React + Vite portfolio Website

Lime Portfolio is a modern, interactive personal portfolio designed to showcase projects, skills, and professional experience. It combines a visually engaging frontend with a functional backend to create a fully dynamic experience for visitors.

## Features

- Responsive design optimized for desktop, tablet, and mobile devices.
- 3D elements and animations for a modern, interactive feel.
- Project showcase with detailed cards for each project.
- Skill bars to highlight technical expertise.
- Contact form powered by an Express.js backend with email integration.
- Downloadable resume directly from the website.
- Smooth navigation across Home, About, Projects, Skills, and Contact sections.

## Tech Stack

- Frontend: React, Vite, CSS (with custom animations and responsive styling)

- Backend: Node.js, Express.js for contact form handling and email functionality

- Hosting: Netlify (frontend) and Render (backend)

## Project Structure

```bash

lime-portfolio/

├─ public/                   # Static assets like resume.pdf

├─ src/

│  ├─ assets/                # Images, icons, 3D models

│  ├─ components/            # Reusable UI components

│  ├─ pages/                 # Page components (Home, About, Projects, etc.)

│  ├─ styles/                # CSS files (global, animations, responsive)

│  ├─ App.jsx

│  └─ main.jsx

├─ server/                   # Backend (Node + Express)

│  └─ index.js

├─ package.json

└─ vite.config.js

```
## Purpose

This portfolio is designed to give potential employers or collaborators a comprehensive view of your skills, projects, and professional experience in an engaging, interactive way. It demonstrates full-stack capabilities by combining a React frontend with an Express backend.
